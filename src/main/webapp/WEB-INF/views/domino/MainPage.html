Domino game

Domino is a game in which a chain of knuckles ("bones", "stones") touching halves with the same number of points is built. The main point of the game is to get rid of your knuckles.

Rules of domino game

Two people are playing. Seven knuckles are handed over, the rest are placed in a closed reserve ("bazaar"). The player who has a 1:1 double in his hands starts, and if no one has such a double, any other in ascending order. If no one has a double, then there is a retake.

The players put up the dice with the corresponding points (1:2, 1:3, 1:*, ...). If a player does not have suitable dice for the next move, he has to look for it in reserve: he chooses one domino "from the bazaar" and if he cannot put it down, then the next one walks.

The game ends when one of the players lays out the last bone, that is, he will not have a single domino left in his hands.
